# QQ红包模拟器

__注意：这只是一个练习程序。__

作者尝试实现QQ拼手气红包，QQ拼手气红包具有这些特点：

- 金额数为两位小数；
- 每个人都可以领到钱；
- 每个红包的金额随机分配；
- 所有红包的金额数总和等于预设的金额。

本程序仅可运行于Python3.2以上的版本，程序没有判断输入是否合法，如果输入意外的数值，将得到不合理的结果。
